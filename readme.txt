Com o Node instalado:

1. Executar linha de comando 'npm install' no diretório do projeto para baixar as dependencias
	Será baixado os diretórios ".nuxt" e "node_modules"

2. Executar o comando 'npm run dev'
	No navegador acessar http://localhost:3000/ (ou a porta que for informada no temrinal)

3. Prontinho! Boa sorte no sorteio! :)
